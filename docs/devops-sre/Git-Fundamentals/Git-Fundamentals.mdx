Git adalah fondasi dari workflow DevOps dan SRE modern. Bagian ini memberikan gambaran umum tentang bagaimana Git digunakan untuk mengelola perubahan, 
berkolaborasi antar tim, serta berperan sebagai single source of truth untuk kode maupun konfigurasi. 
Bagian ini juga memperkenalkan repository Git menggunakan GitLab dan menjelaskan bagaimana GitOps memperluas penggunaan Git ke dalam otomatisasi deployment dan operasional.

## Gambaran Umum

* [**Git**](/docs/devops-sre/Git-Fundamentals/Git)<br />
Menjelaskan konsep dasar version control serta bagaimana Git digunakan untuk melacak perubahan, berkolaborasi dengan tim, dan mengelola histori kode secara aman.

* [**GitRepo**](/docs/devops-sre/Git-Fundamentals/GitLab)<br />
Memperkenalkan bagaimana repository Git (menggunakan GitLab) digunakan untuk menyimpan kode, mengatur akses, melakukan review perubahan, dan mendukung kolaborasi tim.

* [**GitOps**](/docs/devops-sre/Git-Fundamentals/GitOps)<br />
Menjelaskan bagaimana Git digunakan sebagai single source of truth untuk mengotomatiskan deployment dan operasional melalui CI/CD dan konfigurasi deklaratif.
