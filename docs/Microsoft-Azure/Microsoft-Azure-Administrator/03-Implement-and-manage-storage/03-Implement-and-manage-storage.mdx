## Overview

**Azure Storage** adalah solusi penyimpanan cloud dari Microsoft yang dapat menangani lalu lintas tinggi, memastikan ketahanan data serta 
memulihkan data dengan cepat jika terjadi gangguan.
Modul ini membahas bagaimana cara mengelola **storage accounts**, data pada pada **Azure Storage**, **Azure File** dan **Blob Storage**.




* [**Konfigurasi Storage Account**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Implement-and-manage-storage/AZ-1055)<br />
Membahas konfigurasi **storage accounts**, termasuk replikasi dan endpoint.

* [**Konfigurasi Azure Blob Storage**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Implement-and-manage-storage/AZ-1056)<br />
Membahas konfigurasi **Azure Blob Storage**, termasuk tier dan object replication.

* [**Konfigurasi keamanan Azure Storage**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Implement-and-manage-storage/AZ-1057)<br />
Membahas konfigurasi fitur keamanan dasar untuk **Azure Storage** seperti storage access signatures.

* [**Konfigurasi Azure Files**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Implement-and-manage-storage/AZ-1058)<br />
Membahas konfigurasi **Azure Files** dan **Azure File Sync**.
