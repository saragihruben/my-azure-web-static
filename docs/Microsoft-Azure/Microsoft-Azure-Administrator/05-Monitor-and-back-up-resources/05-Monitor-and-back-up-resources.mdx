## Overview

* [**Azure Backup**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Monitor-and-back-up-resources/AZ-1064)<br />
Pengenalan Azure Backup sebagai solusis untuk perlindungan data pada Azure Cloud.

* [**Perlindungan VM menggunakan Azure Backup**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Monitor-and-back-up-resources/AZ-1065)<br />
Pembahasan Azure Backup untuk melindungi on-premise server , virtual machine, SQL server, Azure file share dan beban kerja lainnya.

* [**Pengenalan Azure Monitor**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Monitor-and-back-up-resources/AZ-1066)<br />
Mempelajari bagaimana Azure Monitor menyediakan insight terhadap performa Azure resource dan operasinya.

* [**Meningkatkan incident response dengan Azure Monitor alerts**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Monitor-and-back-up-resources/AZ-1067)<br />
Memberi respon terhadap insiden dan aktivitas pada infrastruktur melalui kemampuan alerting pada Azure Monitor.

* [**Analisa Azure Infrastucture menggunakan Azure monitor logs**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Monitor-and-back-up-resources/AZ-1068)<br />
Membahas penggunaan Azure Monitor logs untuk melakukan ekstrak informasi penting yang tentang infrastruktur dari log data.
