## Overview

* [**Pengenalan Azure Virtual Machine**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1059)<br />
Membahas tentang mesin virtual, serta ekstensi dan layanan yang di gunakan untuk mengelola mesin virtual.

* [**Konfigurasi ketersedian Azure VM**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1060)<br />
Pembahasan konfigurasi ketersediaan **(availability)** pada mesin virtual, termasuk penskalaan **(scalability)** vertikal dan horizontal.

* [**Konfigurasi perencanaan Azure App Services**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1061)<br />
Pembahasan konfigurasi Azure App Service plan , termasuk menentukan *pricing* dan *scaling*.

* [**Konfigurasi Azure App Services**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1062)<br />
Pembahasan konfigurasi dan monitor terhadap Azure App Service instances, termasuk *deployment slots*.

* [**Konfigurasi Azure Container Instance**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1063)<br />
Pembahasan konfigurasi Azure Container Instances termasuk *container groups*.
