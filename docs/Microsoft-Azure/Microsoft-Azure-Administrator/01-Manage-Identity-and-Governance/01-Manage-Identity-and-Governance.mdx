## Overview

* [**Memahami Microsoft Entra ID**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1041)<br />
Pembahasan seputar **Microsoft Entra ID**. Perbandingan **Microsoft Entra ID** dengan **Active Directory DS**, mempelajari Microsoft Entra ID P1 dan P2, 
serta mengeksplorasi Microsoft Entra Domain Services untuk mengelola perangkat dan aplikasi yang terhubung ke domain di cloud.

* [**Pembuatan, konfigurasi dan manajemen identitas**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1042)<br />
Akses pada layanan cloud perlu dikendalikan secara terpusat dengan menerapkan aturan dan kontrol yang jelas untuk setiap pengguna dan resources. 

* [**Komponen arsitektural inti pada Azure**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1043)<br />
Pembahasan komponen  infrastruktur, cara pengelolaan resources, dan pengelolaan resources pada Azure.

* [**Azure Policy Initiatives**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1044)<br />
Pembahasan tentang **Azure Policy Initiatives** dapat digunakan untuk memperjelas standar organisasi dan
evaluasi compliance untuk mengelola Azure resources secara efektif dalam skala besar.

* [**Mengamankan Azure resources dengan Role-Based Access Control (RBAC)**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1045)<br />
Pembahasan tentang implementasi **Azure RBAC** dalam mengelola resources pada Azure.

* [**Reset password menggunakan self-service password pada Microsoft Entra**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Manage-Identity-and-Governance/AZ-1046)<br />
Fitur reset kata sandi mandiri **(self-service password reset)** untuk memungkinkan pengguna mengatur ulang kata sandi atau membuka kunci akun secara mandiri.
