## Overview

* [**Konfigurasi Virtual Network**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1047)<br />
Membahas konfigurasi pada virtual networks dan subnet , termasuk pengalamatan IP.

* [**Konfigurasi Network Security Group**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1048)<br />
Membahas bagaimana implementasi network security groups dan memastikan rulenya diterapkan dengan tepat.

* [**Penggunaan domain pada Azure DNS**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1049)<br />
Membahas pembuatan sebuah DNS Zone pada domain, pembuata DNS records untuk memetakan domain terhadap IP address dan melakukann testing domain untuk resolve terhadap resources.

* [**Konfigurasi Azure Virtual Network peering**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1050)<br />
Membahas konfigurasi peering pada virtual network dan address transit.

* [**Manajemen dan kontrol alur trafik pada Azure deployment menggunakan route**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1051)<br />
Membahas melakukan kendali terhadap virtual network trafik dengan mengimplementasikan custom routes.

* [**Pengenalan Azure Load Balancer**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1052)<br />
Membahas cara kerja Azure Load Balacer dan sebagai solusi untuk di implementasikan sesuai dengan kebutuhan. 

* [**Pengenalan Azure Application Gateway**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1053)<br />
Membahas cata kerja Azure Application Gateway dan sebagai solusi untuk di implementasikan sesuai dengan kebutuhan. 

* [**Pengenalan Azure Network Watcher**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Configure-and-manage-virtual-networks/AZ-1054)<br />
Membahas Azure Network Watcher dalam mendeteksi dan memonitor isu yang berkaitan dengan performansi jaringan pada IaaS resource, 
serta sebagai solusi untuk di implementasikan sesuai dengan kebutuhan.
