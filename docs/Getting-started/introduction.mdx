[MyWiki](http://localhost:3000) adalah personal wiki dari **Ruben Saragih** , seorang yang bekerja di bidang IT Infrastructure. Wiki ini berisi beberapa materi dan hands-on seputar **Cloud Computing** berdasarkan interest dan pengalaman penulis.

## Overview

* [**Microsoft Azure Fundamental**](http://localhost:3000/docs/Microsoft%20Azure/Microsoft%20Azure%20Fundamentals)<br />
Pembahasan seputar Azure Cloud yang menjelaskan tentang **cloud concept**, **architecture**, **services**, **management and governance** pada Azure Cloud. Materi ini mengacu pada Azure Learning untuk completion sertifikasi **AZ-900**.

* [**Microsoft Administrator**](http://localhost:3000/docs/Microsoft%20Azure/Microsoft%20Azure%20Administrator)<br />
Pembahasan seputar Administrasi pada Azure Cloud yang menjelaskan tentang **manage identities** , **governance**, **resources**, **networking**, **storage** dan **monitor and backup**. Materi ini mengacu pada Azure Learning untuk completion sertifikasi **AZ-104**.

* [**Microsoft Azure Hands-On**](http://localhost:3000/docs/Microsoft%20Azure/Microsoft%20Azure%20Hands-On)<br />
Kumpulan hands-on pada Azure Cloud. Penerapan akan lebih banyak menggunakan Infrastructure as a Code (IaaC).

* [**DevOps & SRE**](http://localhost:3000/docs/devops-sre)<br/ >
Pembahasan seputar DevOps & SRE practices.

* [**Others**](http://localhost:3000/docs/others)<br />
Pembahasan lain yang berhubungan dengan IT Infrastructure.
