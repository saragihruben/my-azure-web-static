Azure Administrator

Pembahasan seputar Administrasi pada Azure Cloud yang menjelaskan tentang manage identities , governance, resources, networking, storage dan monitor and backup. Materi ini mengacu pada Azure Learning untuk completion sertifikasi AZ-104.

Overview​

  • Manage Identity and Governance
    Pembahasan tentang manajemen user, akses terhadap resources serta aturan yang dapat diterapkan dalam mengelola layanan pada cloud.

  • Configure and manage virtual networks
    Pembahasan tentang komponen jaringan pada Azure beserta dengan konfigurasi dan penerapannya.

  • Implement and manage storage
    Pembahasan tentang manajemen storage pada Azure.

  • Deploy and manage compute resources
    Pembahasan tentang Azure compute resources beserta dengan konfigurasi dan penerapannya.

  • Monitor and back up resources
    Pembahasan tentang Azure backup dan Azure monitor serta penerapannya.
