## Overview

* [**Introduction to Azure Virtual Machines**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1059)<br />
Covers virtual machines, as well as extensions and services used to manage virtual machines.

* [**Configure Azure VM availability**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1060)<br />
Learn to configuring **availability** in virtual machines, including vertical and horizontal **scalability**.

* [**Configure Azure App Service plans**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1061)<br />
Learn to configuring Azure App Service plans, including selecting the appropriate *pricing* and *scaling* options.

* [**Configure Azure App Services**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1062)<br />
Learn to configuring and monitoring Azure App Service instances, including *deployment slots*.

* [**Configure Azure Container Instances**](/docs/Microsoft-Azure/Microsoft-Azure-Administrator/Deploy-and-manage-compute-resources/AZ-1063)<br />
Learn to configuring Azure Container Instances, including *container groups*.
