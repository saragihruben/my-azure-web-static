title: Azure Fundamentals
---

Pembahasan seputar Azure Cloud yang menjelaskan tentang **cloud concept**, **architecture**, **services**, **management and governance** pada Azure Cloud. Materi ini mengacu pada Azure Learning untuk completion sertifikasi **AZ-900**.


## Overview

* [**Definisi Cloud Computing**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-901)<br />
Pemahaman dasar mengenai cloud computing.

* [**Keuntungan menggunakan Cloud Services**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-902)<br />
Beberapa keuntungan yang bisa di dapatkan saat menggunakan cloud service diantaranya **availability**, **scalability**, **reliability**, **predictability**, **security**, **governance** and **manageability**.

* [**Kategori Cloud Services**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-903)<br />
Pembahasan tentang cloud services yang di kategorikan sebagai **infrastructure as a service (IaaS)**, **platform as a service (PaaS)**, dan **software as a service (IaaS)**.

* [**Komponen Arsitektur dari Azure Cloud**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-904)<br />
Pembahasan Azure cloud secara **physical infrastructure** dan **management infrastructure**.

* [**Azure network and compute services**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-905)<br />
Pembahasan tentang cloud service dan jaringan di Azure. Terdapat tiga opsi komputasi, yaitu **mesin virtual**, **container**, dan **Azure Functions**. 
Untuk jaringan pada Azure, beberapa fitur akan di bahas secara umum seperti **Azure Virtual Networks**, **Azure DNS**, dan **Azure ExpressRoute**.

* [**Azure identity, access, and security**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-906)<br />
Pembahasan tentang **Azure directory service**, **authentication methods**, **external identities**, **conditional access**, **RBAC (Role-Based Access Control)**, **Zero Trust model** , dan penerapan security lainnya.

* [**Azure storage services**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-907)<br />
Pembahasan tentang **Azure storage accounts**, **redudancy**, **services** dan **tools** yang dapat digunakan untuk migrasi dan pemindahan data.

* [**Cost Management pada Azure Cloud**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-908)<br />
Pembahasan tentang faktor yang menentukan pricing, dan tools untuk perencanaan dan estimasi cost dari penggunaan service pada Azure Cloud.

* [**Fitur dan tools pada Azure Cloud untuk penerapan governance and compliance**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-909)<br />
Pembahasan secara umum tentang **Microsoft Purview**, **Azure Policy**, **Resources locks** dan **Service Trust Portal**.

* [**Fitur dan tools pada Azure Cloud untuk managing dan deploying Azure resources**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-910)<br />
Pembahasan secara umum tentang tools yang digunakan dalam managing dan deploying resources pada azure diantaranya **Azure Portal**, **Azure Cloud Shell (termasuk Azure CLI dan PowerShell)**, **Azure Arc** dan **Azure Resources Manager(ARM) & Bicep**.

* [**Monitoring tools pada Azure Cloud**](/docs/Microsoft-Azure/Azure-Fundamentals/AZ-911)<br />
Pembahasan seputar tools yang digunakan pada Azure untuk memberikan insight seperti **Azure Advisor** dan **Application Insights**. 
Juga beberapa tools monitoring seperti **Azure Log Analytics**, **Azure Monitor Alerts** dan **Azure Service Health**.
